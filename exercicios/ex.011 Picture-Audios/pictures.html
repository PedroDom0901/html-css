Áudios

Dica de como adquirir um audio gratuito e com direitos autorais:
Se você tem um canal no YouTube, então:

  1. Acesse o Painel de Controle (YouTube Studio);
  2. Clique em Biblioteca de Audio (No canto esquerdo da tela);
  3. Na Biblioteca de Audios do YouTube, clique em:
    1. Atribuição:
      • Atribuição Necessária: Neste caso, além do áudio que for inserido no seu site, você deve inserir também as linhas exigida;
      • Atribuição não Necessária: Neste caso você está livre de inserir atribuições;
      • Todas as Licenças: Aqui será apresentado todos os áudios com direitos autorais.
    2. Genero;
    3. Duração;
    4. Etc...

Reproduzindo Áudios

Vamos aprender a reproduzir áudio no HTML5.
Ex: <audio src="aqui vai a mídia" controls autoplay></audio>

  • A tag controls serve para possibilitar o úsuario controlar o áudio, isto é, dar play, pausar, retomar, adintar e voltar o áudio.
  • Autoplay serve para reproduzir o áudio automáticamente quando o usuário acessar o site. Isso é útil caso o Dev deseje que o site tenha uma música de fundo.

Obs: Existem três formatos de áudio: MP3,WAV e OGG. Nem todo navegador é compativel com os três. Potanto, neste caso, nós usamos as tags source e type. Ex:

     <audio preload="auto ,metadata ou none" controls loop autoplay>
        <source src="midia/happy-mistake.mp3" type="audio/mpeg">
        <source src="midia/happy-mistake.ogg" type="audio/ogg">
        <source src="midia/happy-mistake.wav" type="audio/wav">
    </audio>

Neste caso, o site tentará reproduzir o áudio com o formato compatível com o navegador do usuário. Caso nenhum dos formatos for compatível, é necessário inserir uma tag para dowload do arquivo.

Parâmetros de Áudio

  1. Preload
    1. auto: é o padrão, porém, é perigoso, porque neste caso o site só será considerado carregado quando carregar todo o áudio também, e geralmente, quando se trata de áudios, os arquivos são pesados.
    2. metadata: será carregado apenas alguns dados como arquivo, nome do arquivo, duração e etc...
    3. none: só será carregado qundo o usuário dar play nos controles.
  2. Controls: Torna os controles do arquivo visíveis no site. Controles do tipo, pausar, retomar, adiantar e voltar.
  3. Loop: Faz o áudio reproduzir automático quando finalizado.
  4. Autoplay: Inicia a reprodução automática ao carregar a página.
