Trabalhando com listas

Lista Ordenada

Para criar uma lista ordenada utiliza-se a tag <ol> como pai e <li> como tag filha, ficando assim:

 <ol type="1" start="1"> 
    <li>Acordar</li>
    <li>Trabalhar</li>
    <li>Estudar</li>
    <li>Dormir</li>
 </ol>

Este é o resultado:

  1. Acordar;
  2. Trabalhar;
  3. Estudar;
  4. Dormir.

Lista Desordenada

A lista desordenada segue o mesmo pricipio, mudando apenas a tag <ol> por <ul>.

  • Pão;
  • Café;
  • Açucar.

Listas de Definições

Nas Listas de Definições usa-se três tags: <dl> , <dt> , <dd> .
Ficando assim:

    <dl>
        <dt>Termo</dt>
        <dd>Descrição</dd>
    </dl>

Exemplo:

HTML:
Linguagem de Marcação de Texto para Sites.
CSS:
Linguagem de EStilos para Sites.
JavaScript:
Linguagem de programação para interatividades no site.
