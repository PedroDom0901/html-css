Principais Formatações

Negrito/Destaque

Nessa frase temos um termo em negritousando a tag <b> (não semantica).

Nesta frase , tenho um termo em destaque usando a tag <strong> (semantica).

Itálico / Ênfase

Nesta frase contem um termo em itálico utilizando a tag <i> (não semantico).

Nesta frase temos um termo em ênfase usando a tag <em> (semantico).

Texto Marcado

Podemos criar tambem um texto marcado usando a tag <mark>.

Texto Grande e Pequeno

Estamos criando texto grande utilizando a tag <big> (obsoleta) e texto pequeno utilizando a tag <small>,

Texto Deletado

Podemos marcar um texto como excluido padra indicar que ele deve ser lido, mas não considerado, utilazando a tag <del>.

Texto Inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois, utilizando a tag <ins>.

Existem tambem os textos sublinhados com a tag <u> (não semantico).

Texto Sobrescrito

Para inserir coisas do tipo x2+3 usando a tag <sup>.

Texto Subescrito

Para inserir coisas do tipo H2O usando a tag <sub>.
