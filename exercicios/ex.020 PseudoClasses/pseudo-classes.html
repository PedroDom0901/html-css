Pseudo-classes

O que é!

As Pseudo-classes são uma mudança/ação gerada em um elemento do seu site ao ato de interação do internalta.

No caso de um id ou classe usamos # e . respectivamente. No caso de uma Pseudo-classe utilize dois pontos (:). E geralmente deve estar relacionada a um elemento ou uma classe. Por exemplo, utilizando um elemento <div> , e uma Pseudo-classe hover, este seria o resultado: div:hover. E deve ser usado num arquivo CSS, seja ele interno ou externo.

A Pseudo-classe hover indica uma mudança em um determinado elemento do seu site. Por exemplo, observe esse três botões abaixo.

Para entender melhor

Para realizar essa ação, utiliamos (>) + a tag do elemento, ficando :

Primeiro passo:
    .texto-escondido > p {
          display : none;
    }
Segundo passo:
    .texto-escondido : hover  > p { 
          display: block;
          elemento 2
          elemento 3
          elemento 4
    }

Onde .texto-escondido é uma classe criada no HTML , hover é a função, > simboliza que o p é filho, ou seja, o parágrafo que esta dentro da classe .texto-escondido.

Observe que no Primeiro passo é utilizado .texto-escondido > p e no Segundo foi acrescentado apenas o hover , pois é no segundo passo que a mágica acontece, porém, isso não quer dizer que o primeiro não seja importante, pois sem ele, a função não fuinciona, e é importante que os primeiros elementos sejam usados conforme no exemplo, sendo:
Primeiro passo: display : none;
Segundo passo: display: block;
